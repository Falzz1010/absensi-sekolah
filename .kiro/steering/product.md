# Product Overview

## Sistem Absensi Sekolah (School Attendance System)

A real-time web-based school attendance management system built for Indonesian schools. The application manages student attendance tracking, reporting, and analytics with multi-role access control.

## Core Features

- **Real-time Dashboard**: Auto-refreshing statistics, attendance charts, and class summaries
- **Multi-role System**: Admin, Guru (Teacher), and Murid (Student) roles with permission-based access
- **Attendance Management**: Individual and bulk class attendance input with QR code scanning
- **Student Portal**: Dedicated panel for students to scan QR codes, submit absence requests, and view attendance history
- **Reporting**: Excel/PDF exports with flexible filtering (date, class, teacher, status)
- **Real-time Notifications**: WebSocket-based instant notifications for attendance events
- **Data Import/Export**: Bulk import teachers and students via Excel templates

## User Roles

- **Admin**: Full system access, user management, data configuration
- **Guru (Teacher)**: Attendance input, schedule management, reporting
- **Murid (Student)**: QR code scanning, absence submission, attendance history viewing

## Attendance Statuses

- Hadir (Present)
- Sakit (Sick)
- Izin (Excused)
- Alfa (Absent)

## Language

The application is in **Bahasa Indonesia** (Indonesian language). All UI text, documentation, and database content use Indonesian terminology.
